Voici le projet labyrinthe que j'ai crée.

Le but du jeux est de sortir du labyrinthe.
Vous pourrez vous déplacer que vers le nord/sud/est/ouest.
Il y a un système de clé qui permettra de débloquer les chemin qui permet de sortir du labyrinthe.
Pour avoir les clé, il faudra les trouvé dans le labyrinthe.
